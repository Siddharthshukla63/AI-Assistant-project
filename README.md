# AI-Assistant-project

## Tic-Tac-Toe GUI (GUIapplication.py)

This project includes a GUI-based Tic-Tac-Toe game built with Tkinter.

### Run

1. Ensure Python is installed (3.8+ recommended)
2. Run from command line:

```
cd c:\project1.py
python GUIapplication.py
```

### Modes

- Player vs Player (`Player vs Player`) 
- Player vs Computer (`Player vs Computer`), computer starts with X in center

### Controls

- Click a cell to place your marker
- Press `Restart Game` to restart

### Notes

- Human player is O in both modes.
- Computer uses random available moves in PvC mode.
